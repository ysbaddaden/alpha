HTML5: <datalist>

This example uses the official compatibility fallback, which is the inclusion of a select element between the datalist and it's option elements.

Please note that a datalist element is quite stupid: it does no filtering, nor any autocompletions: it limits itself to display the list of options, allowing the user to select an entry. But if options are changed the list will dynamically change, allowing to build autocompleters right on top.
